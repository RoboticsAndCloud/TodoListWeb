## Getting started

### Environment
- node v16.20.1
- npm v9.2.0

### Installation

```bash
$ git clone https://github.com/RoboticsAndCloud/eladmin_web_react.git
$ cd eladmin_web_react
$ npm install

$ npm run dev   # URL: http://localhost:5173
```
